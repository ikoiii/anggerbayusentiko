Tugas 10 - IFRAME Implementation

Embedding External Websites

Implementasi IFRAME untuk menampilkan website eksternal dalam halaman web dengan desain yang modern dan interaktif.

Penggunaan IFRAME

Berikut contoh penggunaan IFRAME untuk menampilkan website lain di dalam halaman ini. IFRAME memungkinkan kita untuk mengembed konten eksternal tanpa meninggalkan halaman utama.

Informasi IFRAME:

• Website yang ditampilkan: https://ikoiii.github.io/porto_html/

• Teknologi: HTML5 IFRAME element

• Keamanan: Menggunakan sandbox attributes untuk keamanan tambahan

• Responsive: Otomatis menyesuaikan ukuran layar
