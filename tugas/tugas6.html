Tingkat Hirarki

  1. Hardware

    Komponen fisik yang dapat dilihat dan disentuh dalam sistem komputer

    1. Input Device

      Perangkat untuk memasukkan data ke dalam sistem

      • Mouse
      • Keyboard
      • Joystick
      • Scanner
      • Webcam

    2. Output Device

      Perangkat untuk menampilkan hasil pengolahan data

      1. Monitor
      2. Printer
      3. Speaker
      4. Projector

    3. Process

      Unit pemrosesan utama sistem komputer

      • Central Processing Unit (CPU)
      • Graphics Processing Unit (GPU)
      • Memory Controller

  2. Software

    Program-program yang menjalankan fungsi tertentu dalam sistem

    1. Operating System

      Perangkat lunak dasar yang mengatur hardware

      • Windows
      • Unix
      • Linux
      • macOS

    2. Programming Language

      Bahasa untuk membuat aplikasi dan program

      • Delphi
      • Cobol
      • RPG
      • Python
      • JavaScript

    3. Utility Program

      Program pendukung untuk maintenance sistem

      • Antivirus
      • Disk Cleanup
      • System Monitor
      • File Manager
