Tugas 4 - Pemutar Video Interaktif

Interactive HTML5 Video Player

Pemutar video interaktif dengan kontrol custom untuk play/pause dan pengaturan ukuran video menggunakan JavaScript.

Pemutar Video Interaktif

Klik tombol di bawah untuk mengontrol video.

Informasi Video:

• Format video: MP4

• Kontrol custom dengan JavaScript

• Ukuran dapat disesuaikan (Small/Normal/Big)

• Kontrol Play/Pause yang tersinkronisasi dengan kontrol default HTML5

Teknologi yang Digunakan:

HTML5 CSS3 JavaScript Video API DOM Manipulation Event Handling
